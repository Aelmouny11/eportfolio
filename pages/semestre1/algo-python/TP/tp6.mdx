## Exercice 3 : Distance de Hamming

### 1 DistanceH


```python
def distanceH(s1:str,s2:str):
    if len(s1)!=len(s2):
        print('{} and {} do not have the same length'.format(s1,s2))
        return -1
    distance=0
    for i in range(len(s1)):
        if s1[i]!=s2[i]:
            distance+=1
    return distance

# Testing
distanceH('aabbcc','xaybzc')

```

def binaire(n):
    binary_str = bin(n)[2:]  # remove the '0b' prefix from the binary string
    return binary_str.zfill(8)

        binaire_str=str(n_temp%2)+binaire_str
