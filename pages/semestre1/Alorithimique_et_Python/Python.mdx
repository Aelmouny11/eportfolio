Variables globales et locales

Les variables définies à l'extérieur d'une fonction sont des variablesglobales. Leur contenu est visible de l'intérieur d'une fonction.

Si l’on veut accéder à une variable globale à l’intérieur d’une fonction, on utilise le mot-clé global en Python.




### Variables globales et locales

De manière générale, si elle n’y est pas explicitement déclarée comme globale, une variable x est locale à la fonction dans le corps de laquelle elle est affectée.

Elle est globale si elle est utilisée dans la fonction sans être affectée ou si elle est déclarée globale.
```py
def f( ):
    global a
    a = a + 1
    c = 2 * a
    return a + b + c
```
Dans cette fonction, a est globale car elle est déclarée comme telle,
b est globale car elle est utilisée mais non affectée et c est locale car
elle est affectée mais n’est pas déclarée globale

##     listes

Les listes sont des collections ordonnées et modifiables d'éléments éventuellement hétérogènes. On peut créer des listes en utilisant des méthodes telles que `list()`, `[]`, range, etc. Pour accéder aux éléments d'une liste, on utilise les mêmes méthodes que pour accéder aux caractères d'une chaîne. Les listes peuvent être parcourues à l'aide de boucles for, `range()` et `len()`. Elles sont modifiables, ce qui permet de construire des listes de grande taille de manière dynamique. On peut utiliser le découpage en tranches (slicing) pour ajouter, supprimer ou remplacer des éléments dans une liste. Les listes sont des objets en Python, ce qui signifie qu'elles supportent un certain nombre de méthodes. On peut effectuer des opérations telles que la concaténation et la multiplication sur les listes. Il est également possible de copier une liste.

**Exemples :**

```py
# Création d'une liste
liste1 = [1, 2, 3, 4, 5]
liste2 = list(range(1, 6))

# Accès aux éléments d'une liste
print(liste1[0])  # Affiche 1
print(liste2[2:4])  # Affiche [3, 4]

# Parcours d'une liste avec une boucle for
for element in liste1:
    print(element)

# Modification d'une liste
liste1[2] = 10
print(liste1)  # Affiche [1, 2, 10, 4, 5]

# Recherche dans une liste
if 3 in liste1:
    print("L'élément 3 est présent dans la liste")
else:
    print("L'élément 3 n'est pas présent dans la liste")

# Copie d'une liste
liste3 = liste1.copy()
```

##     tuples

Les tuples sont des types de données composites similaires aux listes, mais ils sont immuables, c'est-à-dire qu'ils ne peuvent pas être modifiés une fois créés. On peut effectuer sur les tuples toutes les opérations des listes, à l'exception de celles qui permettent la modification. Les tuples sont définis à l'aide de différentes syntaxes et peuvent contenir des valeurs de différents types.

**Exemples :**

```py
# Création d'un tuple
tuple1 = (1, 2, 3)
tuple2 = tuple([4, 5, 6])

# Accès aux éléments d'un tuple
print(tuple1[0])  # Affiche 1
print(tuple2[1:])  # Affiche (5, 6)

# Parcours d'un tuple avec une boucle for
for element in tuple1:
    print(element)

# Recherche dans un tuple
if 3 in tuple1:
    print("L'élément 3 est présent dans le tuple")
else:
    print("L'élément 3 n'est pas présent dans le tuple")
```

##  dictionnaire

Les dictionnaires sont des structures de données composées d'éléments non ordonnés, indexés par des clés. Ils sont modifiables comme les listes et peuvent contenir des valeurs numériques, des chaînes, des listes, des tuples, des dictionnaires, des fonctions, des classes ou des instances. On peut créer un dictionnaire en utilisant la syntaxe `{}` ou en utilisant le constructeur `dict()`. Les dictionnaires permettent de faire des opérations telles que la création, le parcours, la recherche et la modification des éléments.

**Exemples :**

```py

# Création d'un dictionnaire
dictionnaire1 = {'fruit': 'pomme', 'couleur': 'rouge', 'prix': 2.5}
dictionnaire2 = dict(nom='John', age=30, ville='Paris')

# Accès aux éléments d'un dictionnaire
print(dictionnaire1['fruit'])  # Affiche 'pomme'
print(dictionnaire2.get('age'))  # Affiche 30

# Parcours d'un dictionnaire avec une boucle for
for cle, valeur in dictionnaire1.items():
    print(cle, '=', valeur)

# Recherche dans un dictionnaire
if 'prix' in dictionnaire1:
    print("Le prix est présent dans le dictionnaire")

# Modification d'un dictionnaire
dictionnaire1['couleur'] = 'vert'
dictionnaire2['age'] = 35

# Suppression d'un élément d'un dictionnaire
del dictionnaire1['fruit']
dictionnaire2.pop('ville')

# Copie d'un dictionnaire
dictionnaire3 = dictionnaire1.copy()
```

##     ensembles (sets)

Les ensembles, ou sets en anglais, sont des structures de données qui modélisent la notion mathématique d'ensemble. Ils peuvent contenir des valeurs de types différents, mais chaque valeur ne peut apparaître qu'une seule fois dans l'ensemble. Les ensembles sont créés en utilisant la syntaxe {} ou en utilisant le constructeur set(). Les ensembles ne sont pas ordonnés, ce qui signifie qu'on ne peut pas y accéder par un indice. Les ensembles supportent des opérations ensemblistes telles que l'intersection, l'union et la différence.

**Exemples :**

```py
# Création d'un ensemble
ensemble1 = {1, 2, 3, 4, 5}
ensemble2 = set([4, 5, 6, 7])

# Accès aux éléments d'un ensemble (non applicable)

# Parcours d'un ensemble avec une boucle for
for element in ensemble1:
    print(element)

# Opérations sur les ensembles
ensemble_intersection = ensemble1.intersection(ensemble2)
ensemble_union = ensemble1.union(ensemble2)
ensemble_difference = ensemble1.difference(ensemble2)

# Test d'appartenance
if 3 in ensemble1:
    print("L'élément 3 est présent dans l'ensemble")
else:
    print("L'élément 3 n'est pas présent dans l'ensemble")
```
